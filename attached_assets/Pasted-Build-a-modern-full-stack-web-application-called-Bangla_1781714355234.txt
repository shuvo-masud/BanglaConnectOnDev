Build a modern full-stack web application called “BanglaConnect”.

🌍 Concept

BanglaConnect is a global networking platform for Bangladeshi students, mentors, and professionals living around the world. It helps users connect, learn, find mentors, and grow professionally.

👥 User Roles

The system has 3 user types:

1. Student
Can browse mentors and professionals
Can view profiles
Can connect and request guidance
Can edit their profile (skills, interests, education, location)
2. Mentor
Can offer mentorship
Must select specialties (e.g., AI, Web Dev, Career Guidance, etc.)
Can toggle availability (available/unavailable)
Has public profile visible to students
3. Professional
Can showcase profession and experience
Can network with others
Can update professional field and skills
🔐 Authentication System

Use Supabase Auth (or equivalent backend):

Email/password signup
Google OAuth login
Session persistence
Secure logout
Auto profile creation on signup

After signup, each user must have a row in profiles table.

🗄️ Database Structure (Supabase/Postgres)
profiles table:
id (uuid, matches auth.users.id)
full_name (text)
role (student | mentor | professional)
bio (text)
country (text)
city (text)
skills (text array)
interests (text array)
professional_field (text)
profession (text, optional)
specialties (text array, only mentors)
mentor_available (boolean, default false)
avatar_url (text)
created_at (timestamp)
